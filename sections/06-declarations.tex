\section*{Data availability}

This study analyses public human--LLM conversation corpora released by their original data providers: WildChat, LMSYS Chat-1M and ShareChat. Readers should obtain the raw public conversation data from the original releases and comply with the corresponding dataset licences, terms of use and redistribution restrictions. The manuscript does not redistribute raw message text, user identifiers or linked user histories.

Derived data sufficient to verify the reported aggregate figures, tables and statistical summaries are archived on Zenodo at \url{https://doi.org/10.5281/zenodo.20995946}. The same release is also available in the GitHub repository at \url{https://github.com/CinderD/Informal-learning-in-everyday-human-LLM-interaction} (tag \texttt{v0.1.0}, commit \texttt{31e10cfea930}). The release includes the numeric values underlying the main figures, de-identified conversation-level and adjacent-turn analytic labels without raw message text, table source files, confidence intervals, p values, false-discovery-rate q values, model-output summaries, provenance documentation and SHA-256 checksums. Any request for additional derived label files that are not included in the public release is assessed against the original dataset licences and privacy constraints.

\section*{Code availability}

Custom code used to export source data and de-identified label tables, compute statistical analyses and generate figures and tables is archived with the derived-data release on Zenodo at \url{https://doi.org/10.5281/zenodo.20995946} and is also available in the GitHub repository at \url{https://github.com/CinderD/Informal-learning-in-everyday-human-LLM-interaction} (tag \texttt{v0.1.0}, commit \texttt{31e10cfea930}). The repository excludes API keys, raw message text and files that cannot be redistributed under the original corpus terms. Reproducing the full annotation pipeline from raw conversations requires user-provided API credentials and access to the raw public corpora from their original sources.

\section*{Ethics statement}

The study is a secondary analysis of public, naturalistic human--LLM conversation datasets. The authors did not recruit participants, intervene in conversations or attempt to identify users. Analyses are reported at aggregate turn or conversation level, and examples in the manuscript are paraphrased rather than copied from raw logs. Human validation was used to assess annotation quality on reviewed examples and did not involve interaction with the original dataset users.
